Visitor country vs Destination

This matrix diagram visualizes the connections between the country that the user has searched from verses the country they wish to visit. The colour of the blocks represent the continent the country is in and a darker colour indicates that a search combination occurred more frequently. The country data in the dataset has been anonymised, which results in the country names being represented as identifiers rather than actual country names.
